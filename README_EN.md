## **Frequently Asked Questions**

1. **Garbled or blurry text on generated pages**
    - You can select a higher output resolution (OpenAI format may not support increasing resolution; Gemini format is recommended). Based on testing, increasing the resolution from 1k to 2k before page generation significantly improves text rendering quality.
    - Please ensure the page description includes the specific text content to be rendered.

2. **Poor results when exporting editable PPT (e.g., overlapping text, missing styles, etc.)**
    - In 90% of cases, this is due to an API configuration issue. Please refer to the troubleshooting and solutions in [issue 121](https://github.com/Anionex/banana-slides/issues/121).

3. **Is the free tier Gemini API Key supported?**
    - The free tier only supports text generation; it does not support image generation.

4. **Encountering 503 error or Retry Error during content generation**
    - You can check the Docker backend logs using the commands in the README to locate the detailed error for the 503 issue. This is typically caused by incorrect model configuration.

5. **Why is the API Key not taking effect after setting it in .env?**
    - The Docker container must be restarted after editing the `.env` file during runtime to apply changes.
    - If parameters were previously configured on the web settings page, they will override the parameters in `.env`. You can restore the `.env` settings by clicking "Restore Default Settings".
